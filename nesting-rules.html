Description

These rules will contain more about nesting . How to nest a code in html in a proper order? and What order should be the right nesting?
There are some common mistakes with solution here.

Some rules

  • The tag opened most recently is always the next tag to close.
  • An HTML element usually consists of a start tag and an end tag, with the content inserted in between.
  • Empty elements do not have an end tag such as (<img>) element.
  • Make sure that you put element inside of another element properly.

This is an example

Do not do this! 

              <html lang="en">
                <head>
                  <meta charset="UTF-8">
                  <meta name="viewport"> content="width=device-width">
                  <title>
                    hhhhhhhhh
                  </head>
                </title>
                </html>

Instead do this 

            <html lang="en">
              <head>
                <meta charset="UTF-8">
                <meta name="viewport"> content="width=device-width">
                <title>
                  hhhhhhhhh
                </title>
              </head>
            </html>
