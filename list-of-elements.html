List of elements in HTML

Definition HTML

HTML stand for HyperText Markup Language. HTML is the standard markup language to create a webpages and for documents designed to be displayed in a web browser.
An HTML element is a type of HTML document component. When you create a document by HTML language, there are several types of Element and one of them HTML which can define (<lang>) attribut. Each element can have HTML attributes specified.
